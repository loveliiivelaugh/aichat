# AiChat

A Progressive Web App featuring a UI to interact with open source LLMs in multiple modalities.

## LLMs

Powered by Ollama

LLM's included:

- LLama3
- Llava
- Mistral

## Modalities

- Text
- Image
- Video
- Code
- Chat
- In Memory Chat
- Web Search
- Vision
- Audio
- Translation

## Features

- Open Source LLMs
- Multiple modalities
- Web UI
